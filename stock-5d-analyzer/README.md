# Stock 5D Analyzer

基于 **5D 分析模型** 的全方位股票分析 Skill，提供从基本面到技术面的深度研判，并根据用户是否持有该资产，输出极具操作性的投资建议（买入区间、止损位、止盈位等）。

## 核心理念

本 Skill 将华尔街高级分析师的研究框架与买方交易的风控逻辑相结合：
1. **5D 深度分析** — 公司基本面 → 技术面 → 宏观行业 → 情绪催化剂 → 风险评估
2. **双场景策略** — 根据用户是否持有资产，分别输出新开仓建议或持仓管理建议
3. **拒绝模糊** — 所有策略必须给出具体价格数字

## 两种使用场景

| 场景 | 用户输入示例 | 输出重点 |
|------|-------------|---------|
| **A：新开仓** | "分析NVDA？" | 投资评级、买入区间、止损位、止盈位 |
| **B：持仓管理** | "持有 100 股 AAPL，成本 180" | 持仓诊断、补仓/减仓建议、动态风控 |

## 使用方法

将 `prompt.md` 的内容加载为 AI Agent 的 System Prompt 即可使用。触发时，Agent 自动按 [Part 1: 5D分析] → [Part 2: 策略结论] 的顺序输出。
