# Stock Value Pick Skill

基于「5D 分析模型 + 价值低估公司筛选」的股票挑选 Skill。

## 核心理念

本 Skill 聚焦 **科技领域及其上下游产业链**，采用华尔街高级分析师级别的 5D 分析模型，筛选 **价值高但当前处于低估状态** 的公司，输出深度投研分析报告。

核心分析流程：
1. **定框架** — 执行候选池构建 → 多维度评分 → 排序精选
2. **深分析** — 5D 模型：基本面 → 技术面 → 宏观行业 → 情绪催化剂 → 风险评估
3. **出报告** — 输出独立、客观的投研分析结论

## 使用方法

将 `prompt.md` 的内容加载为 AI Agent 的 System Prompt/Skill 即可使用。

触发方式仅一种：**启动分析**，Agent 自动执行筛选框架并输出报告。

**输出配置**：
- 输出路径：`/Users/yuanhaoluo/Invest`
- 命名格式：`dsp-YYYYMMDD-模型名.md`
- 示例：`dsp-20260507-gpt4.md`
